# Purpose

This document defines the visual design language of the Muliya Kaizan platform.

Every UI component should follow this design system.

The objective is to create a consistent, scalable and premium user experience.

# Color Rules

Never use more than one accent color inside a component.

Use color only when it communicates meaning.

Do not decorate the interface using colors.

Prefer whitespace over color.

# UI Consistency Rules

Never introduce custom spacing.

Never introduce custom colors.

Never introduce custom typography.

Reuse components whenever possible.

Prefer composition over duplication.

Every page should feel like part of the same product.
